US-009: Video Render Pipeline with BullMQ

IMPLEMENTATION SUMMARY:
- Added Video and RenderJob models to Prisma schema
- Created BullMQ queue system for video rendering jobs
- Implemented 7-stage render pipeline:
  1. Script Generation (10% weight)
  2. Image Generation (25% weight)
  3. Voice Synthesis (15% weight)
  4. Subtitle Generation (10% weight)
  5. Video Assembly (25% weight)
  6. Quality Check (10% weight)
  7. S3 Upload (5% weight)
- Created API endpoints:
  - POST /api/videos/:id/render - Start rendering
  - GET /api/videos/:id/progress - Get progress
- Added comprehensive tests
